  <name>Task 1: Scaffold Vite+React+TS, install dependencies, initialize shadcn</name>
  <read_first>
    - .planning/phases/02-authentication-rbac-user-management/02-RESEARCH.md (Standard Stack — frontend npm commands)
  </read_first>
  <action>
Run from the project root (/d/capstone-hsms):

1. Scaffold Vite project:
   ```bash
   npm create vite@latest frontend -- --template react-ts
   cd frontend && npm install
   ```

2. Install runtime dependencies:
   ```bash
   npm install axios react-router-dom jwt-decode
   npm install -D @types/node
   ```

3. Install IBM Plex fonts (via npm):
   ```bash
   npm install @fontsource/ibm-plex-sans @fontsource/ibm-plex-mono
   ```

4. Initialize shadcn/ui (no preset — UI-SPEC globals.css is authoritative):
   ```bash
   npx shadcn@latest init --defaults
   ```
   When prompted: accept defaults. If interactive, choose: "Default" style, CSS variables for color, "zinc" or any base color (will be overridden by globals.css), yes to CSS variables.

5. Add all shadcn components required by UI-SPEC:
   ```bash
   npx shadcn@latest add button input label dialog alert-dialog table badge skeleton tabs dropdown-menu separator avatar pagination checkbox select sonner alert
   ```

6. Add shadcn blocks:
   ```bash
   npx shadcn@latest add login-01
   npx shadcn@latest add sidebar-07
   ```
   If login-01 is not available as a named block (shadcn blocks are sometimes only available by copying from the website), check with `npx shadcn@latest info`. If unavailable via CLI, note in SUMMARY.md and create the LoginPage manually in Plan 02-06.

7. Configure Vite proxy so `/api` calls are forwarded to the backend.
   Create/update frontend/vite.config.ts:
   ```typescript
   import path from "path";
   import { defineConfig } from "vite";
   import react from "@vitejs/plugin-react";

   export default defineConfig({
     plugins: [react()],
     resolve: {
       alias: {
         "@": path.resolve(__dirname, "./src"),
       },
     },
     server: {
       proxy: {
         "/api": {
           target: "http://localhost:8000",
           changeOrigin: true,
         },
       },
     },
   });
   ```
  </action>
